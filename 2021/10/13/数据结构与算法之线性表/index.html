线性表

线性表，全称是线性存储结构。它的每一个实例都是元素的一个有序集合，每一个元素被线性的的存储在物理内存中。线性表有两种表现形式：顺序存储 和 链式存储。

顺序存储

所有的元素在内存内是连续存储的，类似于数组的形式。

链式存储

所有的元素在内存内是分散存储的，类似于链表的形式。

线性表的顺序存储 - Vector

线性表的顺序存储 Array 的实现，更类似于 C 语言的实现方式。而如果你是在 C++ 的环境中实现，标准模板库 STL 提供了一个基于数组的类 vector ，可以帮助我们更好的实现线性表。

vector 库提供了类似与 ArrayList 的功能。

vector 库中包含了一些功能，包括 erase 和 insert 等。

而且，当数组的内存空间不足时，vector 的容量将按原容量的50% ~ 100%的大小来自动增加。
